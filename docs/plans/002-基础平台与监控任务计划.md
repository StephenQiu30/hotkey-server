# Platform Foundation And Monitor Management Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立 Go API 服务、基础 schema、用户认证和关键词监控任务 CRUD，为后续采集与分析链路提供稳定契约。

**Architecture:** 先实现最小 API 可运行骨架，再落地存储层和认证，再实现监控任务领域对象与 REST 接口。所有入口通过 `internal/server` 统一挂载，存储通过 `internal/store/postgres` 抽象。

**Tech Stack:** Go, net/http or chi, PostgreSQL, SQL migrations, bcrypt, table-driven tests

```go
func (s *Service) Register(ctx context.Context, input RegisterInput) (User, error) {
    if s.repo.ExistsByEmail(ctx, input.Email) {
        return User{}, ErrEmailExists
    }
    hash, err := bcrypt.GenerateFromPassword([]byte(input.Password), bcrypt.DefaultCost)
    if err != nil {
        return User{}, err
    }
    return s.repo.Create(ctx, input.Email, string(hash), input.DisplayName)
}
```
