# 13-B站授权与采集实现计划

## 1. 目标

实现 `BiliBiliFetcher` 适配器，支持视频列表和动态 feed 的标准化采集，处理下架过滤、频控错误和字幕缺失降级。

## 2. 文件清单

- 创建：`migrations/000013_bilibili_source_type.up.sql`
- 创建：`internal/platform/fetcher/bilibili.go`
- 创建：`internal/platform/fetcher/bilibili_test.go`
- 修改：`internal/platform/fetcher/fetcher.go`（注册 bilibili 类型）

## 3. 任务拆解

1. DB 迁移：扩展 `sources.type` CHECK 约束加入 `bilibili`。
2. 定义 `BiliBiliFetcher` 结构体，实现 `Fetcher` 接口。
3. 实现视频列表解析（热门 API 和空间 API 双格式）。
4. 实现动态 feed 解析。
5. 实现下架过滤（code: -404）和频控错误分类（code: -412）。
6. 实现字幕缺失降级到描述字段。
7. 注册 bilibili 来源类型到 fetcher 工厂。

## 4. TDD 与验证

- 视频标准化测试（标题、作者、发布时间、BV ID）。
- 动态标准化测试。
- 字幕缺失降级测试。
- 下架过滤测试（code: -404 返回空列表）。
- 频控错误分类测试（code: -412 返回特定错误）。
- BV/URL 幂等测试。
- HTTP 错误处理测试。
- 空标题跳过测试。

```bash
go test ./internal/platform/fetcher/... -v
go test ./... -count=1
```
