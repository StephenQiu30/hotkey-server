# 03-频道与订阅实现计划

## 1. 目标

提供 AI 频道、用户订阅、用户关键词和日报发送时间偏好。

## 2. 文件清单

- 创建：`migrations/000003_channels_subscriptions.up.sql`
- 创建：`internal/domain/channel/`
- 创建：`internal/domain/subscription/`
- 创建：`internal/service/channel/`
- 创建：`internal/repository/postgres/channelrepo/`
- 创建：`internal/transport/http/handlers/channel/`
- 修改：`internal/transport/http/router.go`

## 3. 任务拆解

1. migration 创建 `channels`、`user_channel_subscriptions`、`user_keywords`、`system_settings`。
2. seed 默认频道：AI 模型、AI 产品、AI 开源、AI 投融资。
3. 用户 API：订阅频道、取消订阅、关键词 CRUD、日报时间设置。
4. 管理 API：频道 CRUD、启停频道。
5. 测试订阅影响日报范围。

## 4. TDD 与验证

- 先写 repository 和 service 测试。
- 再写用户 API 和管理员 API contract 测试。
- 验证普通用户不能修改系统频道。

## 5. 执行顺序

1. `test:` 数据和 service 失败测试。
2. `impl:` migration、repo、service、handler。
3. `docs:` 更新 API 说明或 OpenAPI。

## 6. 回滚策略

回滚 migration 并移除路由，不影响认证模块。

python3 -m unittest discover -s tests
