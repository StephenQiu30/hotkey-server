# 012 B站授权与采集

## 功能目标

支持用户连接自己的 B站授权能力，在授权范围内采集与主题相关的视频、动态、专栏、UP 主信息、发布时间和公开互动指标。

## 用户故事

1. 作为创作者，我可以监控 B站视频和动态中的行业热点。
2. 作为用户，我可以选择关键词或关注频道范围。
3. 作为用户，我希望视频标题、简介和可用字幕进入热点整理。

## 范围

1. 用户授权连接、刷新和撤权。
2. 视频、动态、专栏采集。
3. 标题、简介、可用字幕/转写文本、作者、发布时间、链接、互动指标标准化。
4. 视频下架、字幕缺失、频控和权限限制处理。
5. simulator fixture 和测试账号 smoke。

## 非范围

1. 下载视频二进制。
2. 自动投币、评论、发动态或私信。
3. 绕过地区、版权或登录限制。

## 状态模型

内容文本状态：`metadata_only`、`transcript_available`、`transcript_missing`、`video_unavailable`。

## 契约影响

平台能力必须说明视频文本来源、动态支持、互动指标支持和不可用原因。normalized content 必须支持视频型内容。

## SDD 要求

1. 先定义视频、动态和专栏字段映射。
2. 先定义字幕缺失时的降级摘要策略。
3. 先定义频控和视频不可用失败分类。

## TDD 要求

1. 先写视频和动态标准化测试。
2. 先写字幕缺失降级测试。
3. 先写视频下架不进入事件聚类测试。
4. 先写同一 BV/URL 重复采集幂等测试。

## E2E 验收门禁

1. simulator 返回视频、动态、字幕缺失和下架内容。
2. 视频可用文本进入 AI 摘要链路。
3. 受限状态在 Web 和日报可见。

## 数据与权限

1. 凭据加密存储。
2. 不保存视频二进制。
3. 原文和字幕文本按 30 天留存。
