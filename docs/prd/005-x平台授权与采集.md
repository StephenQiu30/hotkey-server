# X 平台授权与采集 PRD

## 1. 背景

HotKey 需要从 X 平台采集公开内容以丰富热点事件的来源覆盖。X API v2 提供了标准化的 REST 接口，
支持 OAuth 2.0 + PKCE 授权流程，适合服务端安全调用。

## 2. 范围

- X OAuth 2.0 PKCE 授权流程
- 授权撤权
- 关键词相关内容采集（recent search）
- 帖子/作者/发布时间/链接/互动指标/引用关系标准化
- 限流和退避处理
- Schema 变化兼容

## 3. 非目标

- 绕过授权或模拟登录
- 私域内容采集
- 用户时间线采集（需 elevated access）
- DM 或私信采集

## 4. 字段映射

### X Post → source_items 标准化

| X API 字段 | source_items 字段 | 说明 |
|------------|------------------|------|
| `id` | `external_id` | 帖子唯一 ID |
| `text` | `title`, `snippet` | 帖子文本内容 |
| `created_at` | `published_at` | ISO 8601 时间 |
| `author_id` + `id` | `raw_url` | 构造帖子 URL |
| `public_metrics.like_count` | 互动指标 | 存储在扩展字段 |
| `public_metrics.retweet_count` | 互动指标 | 存储在扩展字段 |
| `public_metrics.reply_count` | 互动指标 | 存储在扩展字段 |
| `public_metrics.impression_count` | 互动指标 | 存储在扩展字段 |
| `referenced_tweets[].id` | 引用关系 | 引用/转推的原帖 ID |

### 帖子 URL 构造

```
https://x.com/i/status/{tweet_id}
```

## 5. 限流策略

### X API 限流窗口

| 端点 | 限流（App-level） | 限流（User-level） |
|------|-------------------|-------------------|
| `GET /2/tweets/search/recent` | 450 req/15min | 180 req/15min |

### 退避策略

- 初始退避：1 秒
- 最大退避：15 分钟
- 退避倍数：2（指数退避）
- 429 响应时读取 `x-rate-limit-reset` header 确定重置时间

### 采集频率

- 默认间隔：30 分钟
- 最小间隔：15 分钟
- 可通过 `fetch_interval_min` 配置

## 6. 授权流程

### OAuth 2.0 + PKCE

1. 管理员请求授权 URL（`GET /api/v1/admin/x/auth`）
2. 服务端生成 PKCE code_verifier 和 code_challenge
3. 用户在 X 授权页面完成授权
4. X 回调到服务端（`GET /api/v1/admin/x/auth/callback`）
5. 服务端用 authorization_code + code_verifier 换取 access_token
6. Token 存储在 `x_credentials` 表

### Token 刷新

- 使用 `refresh_token` 自动刷新
- 刷新失败时标记 source 为错误状态
- 支持 `offline.access` scope 获取 refresh_token

### 撤权

- 管理员可撤权（`POST /api/v1/admin/x/auth/revoke`）
- 撤权清除本地存储的 tokens
- 尝试在 X 端 revoke token（best effort）

## 7. 错误处理

| HTTP 状态码 | 处理方式 |
|------------|---------|
| 401/403 | 标记 source 为 auth error，通知管理员 |
| 429 | 指数退避，等待 x-rate-limit-reset 时间 |
| 5xx | 记录错误，下次采集重试 |
| 网络超时 | 记录错误，下次采集重试 |

## 8. 环境变量

| 变量名 | 说明 | 必填 |
|--------|------|------|
| `HOTKEY_X_CLIENT_ID` | X OAuth Client ID | 是 |
| `HOTKEY_X_CLIENT_SECRET` | X OAuth Client Secret | 是 |
| `HOTKEY_X_REDIRECT_URL` | OAuth 回调 URL | 否（默认 localhost） |
