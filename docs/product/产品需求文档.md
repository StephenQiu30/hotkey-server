## 当前 MVP 闭环

P0 闭环为：配置关键词 -> 手动或定时触发 -> AI 查询扩展 -> 多源抓取 -> 候选去重 -> AI 真实性/相关性/重要性分析 -> `active`/`filtered` 阈值过滤 -> 热点展示与即时搜索 -> 事件邮件 -> 日报/周报。

当前已实现的代码路径可覆盖主链路和降级路径；X/Twitter、Bing、SMTP 真实发送和真实模型调用需要配置凭据后单独验收。

企业级扩展路线说明：
- 企业级能力（RBAC、JWT、更多安全与告警、完整可观测性等）不作为当前轻量 MVP 必须项，按 `openspec/changes/enterprise-p0-hotspot-platform/tasks.md` 分批交付。
- 任何企业级扩展项落地前，需先完成本文件定义的 P0 闭环；企业级扩展任务需先在 `docs/engineering/验收标准.md` 明确对应 P0 阶段是否启动，再按 `openspec/changes/enterprise-p0-hotspot-platform/tasks.md` 逐条打开验收清单。
