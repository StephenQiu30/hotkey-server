# Design 文档

本目录存放设计方案类文档。

## 适合放入

1. 技术方案。
2. 架构决策。
3. 接口、数据结构和模块设计。
4. 关键实现权衡。

## 不适合放入

1. 纯产品需求。
2. 执行排期。
3. 测试结果或发布流程。
4. 未形成长期设计决策的临时探索记录。

## 命名建议

使用 `序号-主题-设计.md`，例如 `001-agent角色配置设计.md`。
