# 21-完整事件图谱与跨语言归并验收

## 验收范围

- 事件图谱节点写入：`POST /api/v1/admin/event-graph/events`。
- 事件图谱关系写入：`POST /api/v1/admin/event-graph/relations`。
- 事件图谱查询：`GET /api/v1/events/{id}/graph`。
- 同一事件的中英文内容可通过 `crossLanguageKey` 归并到同一图谱节点。
- 图谱节点保留语言集合、来源事件 ID 和来源事件簇 ID。
- 事件之间可表达关系：
  - `evolution`
  - `citation`
  - `conflict`

## 非目标

- 不在本任务实现图数据库、知识图谱推理引擎或跨服务图谱同步。
- 不依赖未授权来源或旧 FastAPI 图谱结构。
- 不把传播路径和事实冲突仲裁全部纳入本任务；该范围属于任务 22。

## 验收命令

go test ./...

## 验收结果

- 服务层测试覆盖中英文事件按 `crossLanguageKey` 归并。
- 服务层测试覆盖 `evolution`、`citation`、`conflict` 三类关系。
- 服务层测试覆盖非法关系类型拒绝。
- HTTP 测试覆盖跨语言事件节点写入、图谱查询和冲突关系写入。
- OpenAPI 测试覆盖事件图谱节点、关系和查询路径。
