# 20-秒级实时检测验收

## 验收范围

- 授权实时事件推送：`POST /api/v1/realtime/events`。
- 实时源必须登记 source ID 和 token。
- 授权推送可低延迟进入事件候选聚合。
- 实时推送具备窗口限流。
- 实时推送具备熔断状态。
- 限流或熔断时进入降级队列，保留 source、source item、原因和时间。

## 非目标

- 不在本任务接入真实第三方实时流、Webhook 网关或云事件总线。
- 不实现独立实时采集进程、服务网格或自动扩缩容控制器。
- 不绕过外部平台授权，不抓取 token，不做无授权全平台抓取。

## 验收命令

```bash
go test ./...
git diff --check
find docs/product/prd docs/plans -maxdepth 1 -type f | sort -V
```

## 验收结果

- 服务层测试覆盖授权实时推送低延迟进入事件候选。
- 服务层测试覆盖实时源限流后的降级队列。
- 服务层测试覆盖失败熔断后的降级队列。
- 服务层测试覆盖未授权 token 拒绝。
- HTTP 测试覆盖 `POST /api/v1/realtime/events` 成功和限流降级响应。
- OpenAPI 测试覆盖实时事件推送路径。
