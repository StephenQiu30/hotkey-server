# 3-关键词与用户偏好验收

## 1. 范围

本验收对应 GitHub issue `#73`，父 Epic 为 `#67`。

已完成内容：

1. 平台关键词创建、列表、启停接口。
2. 用户关注关键词接口。
3. 用户屏蔽关键词接口。
4. 用户追加关键词接口。
5. 用户偏好查询接口。
6. 结构化错误响应。
7. OpenAPI 路径覆盖。

## 2. 验证命令

```bash
go test ./...
git diff --check
HOTKEY_HTTP_ADDR=127.0.0.1:18081 go run ./cmd/server
curl -fsS -X POST http://127.0.0.1:18081/api/v1/admin/keywords \
  -H 'Content-Type: application/json' \
  -d '{"term":"OpenAI","category":"lab"}'
curl -fsS http://127.0.0.1:18081/api/v1/admin/keywords
curl -fsS -X POST http://127.0.0.1:18081/api/v1/keywords/follow \
  -H 'Content-Type: application/json' \
  -d '{"userId":"user-1","term":"OpenAI"}'
curl -fsS -X POST http://127.0.0.1:18081/api/v1/keywords/block \
  -H 'Content-Type: application/json' \
  -d '{"userId":"user-1","term":"AI slop"}'
curl -fsS -X POST http://127.0.0.1:18081/api/v1/keywords/additional \
  -H 'Content-Type: application/json' \
  -d '{"userId":"user-1","term":"Claude Code"}'
curl -fsS 'http://127.0.0.1:18081/api/v1/keywords/preferences?userId=user-1'
curl -fsS http://127.0.0.1:18081/openapi.json
```

## 3. 验证结果

- `go test ./...` 通过。
- `git diff --check` 通过。
- 平台关键词可创建、列表可读取。
- 用户偏好可写入关注、屏蔽和追加关键词。
- 用户偏好查询返回 `followedKeywords`、`blockedKeywords`、`additionalKeywords`。
- `/openapi.json` 包含关键词管理与用户偏好路径。

## 4. 后续边界

当前 feature 使用进程内仓储锁定 API 行为。PostgreSQL schema、迁移、持久化读写和用户身份鉴权将在后续 P0 任务继续接入。本任务不实现来源采集、内容聚合、pgvector 相似聚合或日报能力。
