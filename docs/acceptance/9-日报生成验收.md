# 9-日报生成验收

## 1. 实现范围

- 新增日报服务，支持生成平台日报和用户关注日报。
- 平台日报按热度排序输出昨日热点条目。
- 用户日报按关注关键词过滤热点条目。
- 日报条目必须回链事件簇 `eventId` 和证据链 `evidenceIds`。
- API 默认查询昨日，也支持显式 `YYYY-MM-DD` 日期。

## 2. API 影响

- `GET /api/v1/reports/daily?date=YYYY-MM-DD`
- `GET /api/v1/users/{id}/reports/daily?date=YYYY-MM-DD&keywords=OpenAI,model`
- `/openapi.json` 已包含平台日报和用户日报接口。

## 3. 验收命令

```bash
go test ./...
git diff --check
find docs/product/prd docs/plans -maxdepth 1 -type f | sort -V
```

## 4. 验收结论

- 每日可生成昨日平台日报。
- 用户可按关注关键词查看日报。
- 每条日报都能追溯事件簇和证据链。
- 当前实现先使用进程内生成器锁定领域和 OpenAPI 契约；后续定时任务、Redis 锁和持久化会在 P0 后续任务中接入。
