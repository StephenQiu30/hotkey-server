## 范围与残余风险

- 已验收的是纯配置控制平面：immutable Monitor config、SourceConnection、审计、preview 与 HTTP 契约。
- 本验收**未执行且不在范围**：外部 fetch/validate/health、Connector、RSS/Atom/HN 网络请求、River、Content、collection run/checkpoint 应用或推进、真实凭据/秘密、Embedding、事件、报告或浏览器联调。`collection_run_targets` 仅作为未来 PLAN-006 的 Schema/记录契约存在。
- 因而残余风险仅属于 PLAN-006：Connector 的 DNS/redirect SSRF 执行防护、条件请求、配额/限流、分页、共享 runs 与 checkpoint 推进必须自行实现并验收；本结论不替代那些门禁。
