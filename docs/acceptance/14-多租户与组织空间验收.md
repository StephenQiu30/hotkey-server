# 14-多租户与组织空间验收

## 验收范围

- 用户可属于多个租户：`POST /api/v1/admin/tenants`、`POST /api/v1/admin/tenants/{id}/members`、`GET /api/v1/users/{id}/tenants`。
- 关键词可按租户隔离：`PlatformKeyword.tenantId` 与 `ListPlatformKeywordsByTenant`。
- 来源可按租户隔离：`Source.tenantId`、`ListSourcesByTenant` 与 `UpdateTenantSourceConfig`。
- 日报可按租户隔离：`GET /api/v1/tenants/{id}/reports/daily`。
- OpenAPI 导出包含租户管理、用户租户列表和租户日报契约。

## 非目标

- 不在本任务实现复杂 RBAC、审计日志或租户级后台配置扩展。
- 不引入多服务拆分、真实计费或完整消息队列。
- 不恢复旧 FastAPI 运行时或旧数据库结构。

## 验收命令

go test ./...

## 验收结果

- 租户服务测试覆盖创建租户、添加成员和单用户多租户归属。
- 关键词、来源、日报测试覆盖租户级隔离。
- HTTP 测试覆盖租户创建、成员添加、用户租户列表和租户日报。
- OpenAPI 测试覆盖 P1 租户契约路径和状态码。
